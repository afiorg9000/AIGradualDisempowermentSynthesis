# AI Gradual Disempowerment Synthesis

A living synthesis of research on gradual AI disempowerment — the progressive erosion of human oversight capacity, agency, and institutional control relative to AI systems.

## What this is

A single-file interactive website covering 2019–2026, with 30 primary sources, trajectory visualization, convergence pattern analysis, 8 research gaps, and a live source database connected to Airtable.

## Structure

- `index.html` — the complete site (self-contained, no dependencies)

## Sections

1. **Argument** — 5-step case for gradual disempowerment
2. **Mechanism** — the self-concealing feedback loop (5 stages with empirical evidence)
3. **Trajectory** — interactive timeline 2019→threshold (9 clickable phases)
4. **Convergence** — 7 patterns across independent research teams
5. **Gaps** — 8 critical open questions
6. **Living Synthesis Database** — 30 sources, searchable, Airtable-connected

## Setup

Open `index.html` directly in a browser — no server required.

To enable Airtable submission: replace `AIRTABLE_API_KEY` in `index.html` with your personal access token from [airtable.com/create/tokens](https://airtable.com/create/tokens).
